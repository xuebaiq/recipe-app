# 🚀 快速开始指南

## Windows用户（最简单）

### 第一步：双击启动
```
找到并双击 "启动.bat" 文件
```

### 第二步：等待启动
- 首次运行会自动安装依赖（需要几分钟）
- 看到"服务器启动成功"就可以了

### 第三步：打开浏览器
```
在浏览器地址栏输入：http://localhost:5000
```

## 就这么简单！

## 基本使用

### 1️⃣ 查看今日推荐
- 打开网页自动显示
- 早餐、午餐、晚餐各有推荐
- 点击菜谱卡片查看详细做法

### 2️⃣ 切换饮食模式
- 点击顶部"中餐"按钮可切换为"地中海饮食"
- 中餐：每餐7个选项
- 地中海：每餐4个选项

### 3️⃣ 搜索菜谱
- **搜索蔬菜**：输入"黄瓜"，得到6种做法
- **搜索菜名**：输入"鱼香肉丝"，得到2种做法
- 食材清单在最前面，方便记录

### 4️⃣ 查看详情
- 点击任意菜谱卡片
- 查看完整的食材列表和制作步骤

## 遇到问题？

### ❓ 启动失败
- **没有Python？** 下载安装Python 3.8+（https://python.org）
- **端口被占用？** 关闭其他可能占用5000端口的程序

### ❓ 页面打不开
- 确认命令行窗口显示"服务器启动成功"
- 试试这个地址：http://127.0.0.1:5000
- 检查防火墙是否阻止

### ❓ 字体太小
- 按键盘 Ctrl 和 + 号放大页面
- 或者在浏览器设置中调整缩放

## 高级功能（可选）

### 🔑 启用AI扩展
如果想在本地数据库找不到菜谱时，自动联网查询：

1. **获取新的API Key**
   - 访问：https://siliconflow.cn
   - 注册账号并获取API Key
   - ⚠️ 之前的key已泄露，必须重新生成

2. **设置API Key**
   ```batch
   # Windows（在命令行中运行）
   setx SILICONFLOW_API_KEY "你的新key"
   ```

3. **重启程序**
   - 关闭当前运行的程序
   - 重新双击"启动.bat"

**注意**：不设置API Key也完全可以正常使用本地180道菜谱！

## 💡 小技巧

1. **采购清单**
   - 搜索菜名后，食材列表就是采购清单
   - 可以截图或打印

2. **节日推荐**
   - 系统会根据农历节日自动推荐特色菜
   - 比如春节有饺子，冬至有汤圆

3. **健康饮食**
   - 系统优先推荐低卡、中卡菜谱
   - 适合控制体重

4. **刷新推荐**
   - 点击"刷新推荐"按钮
   - 获取新的随机推荐

## 📱 手机使用

1. **确保手机和电脑在同一WiFi**

2. **查看电脑IP地址**
   - Windows：命令行输入 `ipconfig`
   - 找到"IPv4 地址"，比如 192.168.1.100

3. **在手机浏览器打开**
   ```
   http://192.168.1.100:5000
   （替换为你的实际IP）
   ```
